Starter

Will provide a lamp environment to test some of well known PHP frameworks. Each frame has its own branch in the repository as well as this branch (lamp-frameworks) which provide all the frameworks available in other branches.

If you have resolved predefined domain and sub-domains as explained in the this branch README file, you should be able to visit available frameworks using following urls:

CakePHP
cakephp25.takeawayit.local
cakephp.takeawayit.local
